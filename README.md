# To run app
Step 1 - Locate the "package.json" file and open the terminal

Step 2 - type "npm install"

Step 3 - create a ".env" file with "MONGO_URI", "RAZORPAY_ID_KEY" & "RAZORPAY_SECRET_KEY"

Step 4 - type "npm start"
